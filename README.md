# Text Forge
A script that generates text in Asset Forge
![](https://imgur.com/PFpRmyJ =446x245)

## Installation
- Download/clone the script.
- Open Asset Forge.
- Click on `Scripts` in the upper right corner.
- Open the script.

## Documentation
